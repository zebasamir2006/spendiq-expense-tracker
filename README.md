# 💰 SpendIQ – Smart Expense Tracker & Finance Dashboard

SpendIQ is a modern **Streamlit-based web application** that helps users track, analyze, and visualize personal expenses in a simple and interactive way.  
It provides clear financial insights through analytics and charts, helping users understand their spending habits better.

---

## 🚀 Live Features

### ➕ Expense Management
- Add expenses with title, amount, category, and date
- Store and manage financial records efficiently

### 📋 Data Overview
- View all expenses in a clean table format
- Organized and structured financial data display

### 📊 Analytics Dashboard
- Total expenses overview
- Highest expense tracking
- Lowest expense tracking
- Category-wise spending analysis
- Monthly spending summary

### 📈 Interactive Visualizations
- 📊 Category-wise Bar Chart
- 🥧 Expense Distribution Pie Chart
- 📉 Monthly Spending Trend Line Chart

### ⬇ Export Feature
- Export all expense data to CSV file for external use

---

## 🛠️ Tech Stack

- **Python** 🐍  
- **Streamlit** ⚡ (Web App Framework)  
- **Pandas** 🐼 (Data Handling)  
- **Matplotlib** 📊 (Data Visualization)  
- **SQLite / Custom Modules** 🗄️ (Data Storage)
