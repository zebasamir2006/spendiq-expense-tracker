# 💰 SpendIQ – Smart Expense Tracker & Finance Dashboard

SpendIQ is a modern **Streamlit-based web application** that helps users efficiently manage their daily expenses. It allows users to record transactions, analyze spending patterns, visualize financial data through interactive charts, and export records for future use.

Whether you're tracking everyday spending or exploring your financial habits, SpendIQ provides a simple and user-friendly solution.

---

## ✨ Features

### ➕ Expense Management
- Add expenses with title, amount, category, and date
- Store expense records securely

### 📋 View Expenses
- Display all recorded expenses in a clean table
- Organize spending by category and date

### 📊 Analytics Dashboard
- View total expenses
- Identify highest and lowest expenses
- Category-wise expense analysis
- Monthly expense summary

### 📈 Data Visualization
- 📊 Category-wise Bar Chart
- 🥧 Expense Distribution Pie Chart
- 📉 Monthly Expense Trend Line Chart

### 📤 Export Data
- Export expense records to CSV format

## 🚀 Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/spendiq.git
```

### 2️⃣ Navigate to the project folder

```bash
cd spendiq
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
streamlit run app.py
```
